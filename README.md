# Инструкция если вы на Arch linux

1. открываем терминал

2. устонавливаем sfml такой коммандой sudo pacman -S sfml

3. переходим в папку с игрой

4. вводим комманду chmod +x ./make.sh

5. потом запускаем ./make.sh

6. потом ждём других нормальных проектов от меня
